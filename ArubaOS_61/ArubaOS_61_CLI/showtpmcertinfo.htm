show tpm cert-info

show tpm cert-info

Description

Displays the TPM and Factory Certificate information on MIPS controllers (M3, 3000 Series, 600 Series).

Syntax

No parameters.

Usage Guidelines

Use this command to verify that TPM and factory certificates are installed as expected. This command should be executed before enabling CPSec on MIPS controllers (M3, 3000 Series, 600 Series).
